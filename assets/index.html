How It Works

From Chat to Playable in Four Steps

No setup, no compilation, no coding required. The entire pipeline runs in your browser.

①

Configure

Add your DeepSeek API key (or any OpenAI-compatible endpoint) in settings. BYO-Key architecture — keys never touch our servers.

②

Describe

Tell the agent what game you want. "Make a neon-styled brick breaker" or "Build a space shooter with particle effects".

③

Generate

The agent reads scaffold docs, studies templates, delegates research to subagents, writes code, and builds — all autonomously.

④

Play

The game appears in the preview panel instantly. Not satisfied? Keep chatting to iterate, or start a new session.

SCAFFOLD

Knowledge Base

Authoritative development guides, architecture patterns, 20+ gotcha rules, and four reference game templates (Snake, Breakout, Tetris, 2048). Extensible by the agent itself.

WORKSPACE

Session Isolation

Each session gets an isolated workspace. Four-layer path validation prevents traversal. Agent interacts via 10 tools — file ops, ripgrep search, build, skills, todos, and subagent delegation.

PROVIDER

Model Resilience

OpenAI-compatible SDK with DeepSeek support. Factory pattern for Claude/OpenAI expansion. Built-in fallback: v4-pro primary, v4-flash automatic failover on API errors.
