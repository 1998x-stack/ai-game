核心能力

四大特性，重新定义游戏开发

从想法到可玩的游戏，只需一次对话。无需安装、无需编译、无需编写代码。

💬

自然语言驱动

用中文或英文描述你的游戏创意——"做一个贪吃蛇游戏，霓虹风格"——Agent 立即理解并开始构建。

⚡

实时流式生成

Agent 逐步展示工具调用、推理过程与代码生成，每一行代码都清晰可见，开发过程完全透明。

🎮

即时预览

右侧沙箱 iframe 直接运行生成的游戏，支持键盘、鼠标、触控操作，还可一键全屏体验。

🔄

持续迭代

"加快速度""加个计分板""换色调"——多轮对话不断优化游戏，支持会话恢复，开发永不中断。

使用流程

四步上手，零门槛创造

没有复杂的配置和学习曲线。从零开始，体验 AI 驱动的游戏开发。

1

配置 API Key

在设置中填入你的 DeepSeek API 密钥（或任何兼容 OpenAI 的端点），即可开始使用。

2

描述游戏创意

用自然语言描述你想玩的游戏——例如"做一个打砖块游戏，紫色渐变风格"。

3

Agent 自动生成

AI Agent 读取权威模板和最佳实践，自动生成优化后的完整游戏代码并构建。

4

立即试玩

游戏在预览面板中立即运行。不满意？继续对话调整，或在当前游戏上迭代完善。

SCAFFOLD

知识库

权威开发指南、架构模式、反模式集合（Gotchas）和四种参考游戏模板（贪吃蛇、打砖块、俄罗斯方块、2048）。

WORKSPACE

会话隔离

每次会话拥有独立工作区，路径安全验证防止越权访问。Agent 通过 6 种工具（读/写/编辑/列表/构建/报错）与工作区交互。

PROVIDER

模型扩展

基于 OpenAI 兼容 SDK，现支持 DeepSeek。工厂模式设计预留了 Claude、OpenAI 等扩展接口，可轻松接入更多模型。
